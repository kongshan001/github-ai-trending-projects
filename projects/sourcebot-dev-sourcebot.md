## 💎 核心价值

Sourcebot is a self-hosted tool that helps humans and AI agents search, read, and navigate code across many repositories.

## 📋 业务场景

- 自托管代码搜索（团队/企业内部）
- Sourcegraph 的开源轻量替代
- AI Agent 跨仓库读代码

## 🎯 最佳实践

- **自托管**：完全掌控数据，无需依赖外部 SaaS
- **MCP 暴露**：可作为 MCP 服务被 AI Agent 调用
- **多仓库**：天生支持跨仓库搜索
